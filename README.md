# Whiteboard-Editor
An interactive web whiteboard

How to use it
Choose a tool: Pen, Rect, Circle, Line, or Select.

Draw shapes: Click-drag to create; release to finalize. Fill and stroke controls apply.

Pen tool: Click-drag to freehand draw; width and stroke apply.

Select tool:

Click a shape to select; drag inside to move it.

Drag a handle to resize (rect corners, circle radius, line endpoints).

Drag on empty space to draw a selection box and multi-select.

Grouping: Press G or click Group; Ungroup with U.

Layer ordering: Bring forward ] or send backward [.

Clear: Resets the board.
